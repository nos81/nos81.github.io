# noshnik.com

Personal website of **Nimba Oshnik**.

Live at **[www.noshnik.com](https://www.noshnik.com)** · hosted on GitHub Pages.

## About

A hand-written static site.
Open `index.html` in a browser.

- **Single-page, long-scroll design.** A fixed background, crossfade effect
  between sections.
- **Light / dark theme** with a toggle. The choice persists.
- **Progressive enhancement.** Everything readable with JavaScript disabled; Fully
  obeys `prefers-reduced-motion`.

## Layout
